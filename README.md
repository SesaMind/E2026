

> **First Principles Handbook for Advanced Oral Proficiency**  
> A systematic learning repository designed to help learners achieve **Advanced Low (AL)** or higher in the OPIc English Speaking Test.

---

## Project Vision

Most OPIc preparation materials teach learners **what to answer**.

This repository teaches **why the test asks certain questions**, **how responses are evaluated**, and **how to build natural spoken English from first principles**.

Instead of memorizing hundreds of topic-specific answers, you will learn the underlying mechanisms that generate nearly every OPIc question.

The objective is not to "pass AL."

The objective is to build speaking ability that naturally produces an AL-level performance.

---

# Design Philosophy

This repository follows five principles.

## 1. First Principles

Every lesson begins with the underlying mechanism before introducing techniques or examples.

Questions answered include:

- Why does OPIc ask this type of question?
    
- What speaking skill is actually being evaluated?
    
- What distinguishes AL from IH?
    
- What causes otherwise fluent speakers to lose points?
    

---

## 2. MECE Organization

Every concept is categorized into mutually exclusive, collectively exhaustive groups whenever practical.

The repository minimizes duplication and encourages reusable mental models.

---

## 3. System Thinking

Topics are treated as surface-level variations.

Templates, discourse structures, and communication mechanisms are treated as the true learning objectives.

---

## 4. High Information Density

Every section should provide actionable knowledge.

The repository intentionally avoids:

- filler
    
- motivational writing
    
- repetitive explanations
    
- memorization-based learning
    

---

## 5. Production Quality

Every lesson is written as a permanent reference document rather than temporary notes.

# Learning Principles

This repository is intentionally cumulative.

Do not skip chapters.

Every module builds upon previous concepts.

Recommended order:

1. Foundations
    
2. Scoring System
    
3. Background Survey
    
4. Topic Archetypes
    
5. Question Templates
    
6. Discourse Management
    
7. Speaking Frameworks
    
8. Mock Tests
    

---

# Expected Outcome

Upon completing this repository, learners should be able to:

- understand how OPIc generates questions
    
- recognize question templates immediately
    
- organize responses efficiently
    
- maintain coherent discourse
    
- recover naturally after communication breakdowns
    
- perform consistently at AL or above
    

---

# What This Repository Is Not

This repository is **not**:

- a collection of memorized answers
    
- a vocabulary list
    
- an expression handbook
    
- a shortcut guide
    
- a "guaranteed AL" strategy
    

It is a structured learning system for developing transferable speaking ability.

---

# Target Audience

This handbook is designed for learners who:

- aim for AL or higher
    
- prefer analytical learning
    
- enjoy structured frameworks
    
- value long-term improvement over memorization
    
- want to understand the mechanics behind the exam

# Recommended Study Method

For each lesson:

1. Read the mechanism.
    
2. Study the framework.
    
3. Analyze the examples.
    
4. Complete the exercises.
    
5. Review the checklist.
    
6. Take the quiz.
    
7. Practice speaking aloud.
    
8. Continue to the next lesson only after mastering the current one.
